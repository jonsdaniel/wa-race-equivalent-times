Method
All events are mapped to the same points value, then the matching time for each event is displayed.
Missing cells
If a specific event has a “-” at a given points value, this tool uses the next faster available time (from the next higher points row).
Disclaimer
These equivalences are scoring-based comparisons, not physiology-based predictions. Individual strengths (speed vs endurance), course, weather, and training can shift real-world equivalence.
